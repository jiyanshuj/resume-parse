# Screenshots Folder

This folder contains application screenshots for the README showcase.

## Required Screenshots:

Save your application screenshots with these exact filenames:

1. **upload-screen.png** - Resume upload interface
2. **profile-data.png** - Extracted professional profile with personal information
3. **skills-section.png** - Skills and competencies display
4. **education-projects.png** - Education history and projects section
5. **certifications-links.png** - Certifications and professional links

## How to Add Screenshots:

1. Take screenshots from your mobile application
2. Save them with the exact filenames listed above
3. Place them in this `screenshots` folder
4. The images will automatically appear in the main README.md

## Image Guidelines:

- Format: PNG or JPG
- Recommended width: 300-400px (mobile screenshots)
- Keep file sizes reasonable (< 1MB per image)
- Ensure sensitive information is redacted if needed
